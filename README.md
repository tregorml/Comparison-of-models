## Comparison of 8 baseline models for predicting survival on the Titanic

The project demonstrates the effectiveness of the model in solving the classical ML problem
(Prediction of the number of survivors)

## Features

- Show train every model
- Show test every model
- Show GAP every model
- Show the best model in train
- Show the best model in test
- Show the best model in GAP
